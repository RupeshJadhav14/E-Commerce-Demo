# E-Commerce Website Demo

This is a responsive eCommerce website demo built to showcase modern UI design, product display, and user-friendly shopping experience.

🔗 Live Demo: https://rupeshjadhav14.github.io/E-Commerce-Demo/

---

## 📌 Overview

This project demonstrates a clean and visually appealing eCommerce layout with a focus on usability and responsiveness across devices.

It simulates a real-world online store interface including product listings, navigation, and call-to-action elements.

---

## 🚀 Features

- Responsive design (mobile-friendly)
- Modern UI layout
- Product listing section
- Call-to-action buttons (Add to Cart / Buy Now)
- Clean navigation and structure
- Fast and lightweight

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript
- Bootstrap
